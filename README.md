# Digital-currency-price
This application have all fuctionality to get digital currency price, folders and fils categorise with name,  Arabic language

In this application I used general UITableView and general UICollectionView, also API calls from one place WebService file so
if API link change all you need is change url in BaseRequest file.

Constant file add to this code for strings, default label, colors, cell ID, viewcontroller ID, API parameters, API calls, alert message and button text, fonts name

Extension for what it need like hexColor all you have to do is < "95989C".color_ >
Date extension -> convert date to string and string to date using stringToDate() and for date to custome string using dateToString(customFormat: "dd/MM/yyyy")

UITableView extension to shoe loading using this function with table view like tableView.showLoading() it will appear in tableView footer
